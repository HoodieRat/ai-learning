Image AI Capstone: Build a Style Guide Workflow

Consistency is the real product. You’ll build a style guide and a repeatable workflow that yields a coherent set of images.

Prereqs

  • Image AI: Pitfalls

Artifact

  • A style guide (subject, palette, composition, do/don’t)
  • A prompt template + constraints
  • 6 images that look like one set

Plan your guide

  1. Define subject, palette, lighting, composition, and what to avoid.
  2. Create a base prompt with fixed seed and guidance; add a negative prompt.
  3. List variables you may change (pose, prop, background) one at a time.
  4. Decide how you will judge consistency (3 visual criteria).

Mini build: produce the set

  1. Generate 3 images with the base prompt; pick the best as reference.
  2. Generate 3 more, varying one variable per image.
  3. Apply one targeted fix (inpaint/control) if an image drifts.
  4. Document the final recipe so image #7 can be produced later.

Practice Lab

Task: Produce a coherent mini set (6 images).

Do: Lock one “source of truth” prompt and vary only one variable at a time (pose, setting, prop).

Success criteria:

  • All images share the same style and identity.
  • You document your variables and constraints.
  • You can repeat the workflow to add a 7th image that matches.

Verify: Generate image #7 a day later using only your guide and confirm it matches.

Recap

  • Style guides need fixed seeds/prompts plus a list of allowed changes.
  • Vary one thing at a time; use targeted fixes to maintain consistency.
  • Write down the recipe so future images stay on-style.
